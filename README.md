# ИИ-ассистент для мастерской по пошиву костюмов furrsuit

Ссылка на проект - @FurriyaBot

## Описание проекта

Разработан интеллектуальный ассистент для мастерской по созданию кастомных костюмов furrsuit, который автоматизирует коммуникацию с клиентами, предоставляет подробную информацию об услугах и упрощает процесс записи на консультации.

## Функциональные возможности

### 💬 Осмысленный диалог
Ассистент поддерживает естественные разговоры с клиентами, понимая контекст общения и адаптируясь под стиль коммуникации. Нейросеть способна:
- Отвечать на вопросы о процессе создания костюмов
- Объяснять особенности различных материалов и технологий
- Помогать клиентам определиться с дизайном и стилем
- Поддерживать дружелюбный и профессиональный тон общения

### 📚 База знаний
Система интегрирована с обширной базой знаний, содержащей:
- Информацию о видах furrsuit-костюмов (полные, частичные, квест-костюмы)
- Данные о материалах (мех, ткани, наполнители)
- Описание этапов создания костюма
- Ценовую политику и сроки изготовления
- FAQ по уходу за костюмами
- Примеры работ и портфолио

### 📅 Система записи
Автоматизированный процесс бронирования встреч:
- Проверка доступности свободного времени
- Запись клиента на удобное время
- Автоматическое добавление встречи в Google Календарь
- Отправка подтверждений и напоминаний

## Использованные технологии

### Qwen
**Роль:** Генерация контента и настройка ИИ

Использован для:
- Создания детального системного промпта, определяющего поведение и стиль общения ассистента
- Формирования и структурирования базы знаний
- Настройки тональности ответов (профессиональный, но дружелюбный)
- Оптимизации сценариев диалогов

### Савви (https://suvvy.ai/)
**Роль:** Платформа для создания ИИ-ассистента

Обеспечивает:
- Интуитивный интерфейс для настройки бота
- Интеграцию с различными мессенджерами и платформами
- Подключение к Google Календарю для управления расписанием
- Аналитику взаимодействий с клиентами
- Масштабируемость решения
